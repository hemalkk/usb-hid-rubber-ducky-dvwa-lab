# Ethical Notice

This repository is an educational cybersecurity portfolio project.

The payloads are designed only for a controlled local lab environment using a virtual machine and DVWA. They are included to demonstrate how HID automation works and how defenders can detect and mitigate this behaviour.

## Not Included

This repository does not include:

- Malware
- Credential theft
- Browser data collection
- Password harvesting
- SSH key collection
- Persistence
- Reverse shells
- Destructive commands
- SQL injection
- Real exploitation
- Real data exfiltration

## Authorised Use Only

Only run these payloads on systems that you own or have explicit permission to test.

## Defensive Purpose

The main purpose of this project is to support learning in:

- USB HID attack awareness
- Endpoint detection
- Social engineering risk
- Safe lab simulation
- MITRE ATT&CK mapping
- Security mitigation planning
